# LookItUp
This is an OSINT tool that can be used for information gathering, reconnaissance, and more . 
It can perform numerous tasks like - 

     ▪ Email Validator 
     ▪ Whois Lookup 
     ▪ Geo-location Lookup 
     ▪ What's My Public IP 
     ▪ Port Scanner 
     ▪ URL Expander
     ▪ URL Safety Check
     ▪ MAC Address Lookup
